Add Bookmarks via Git

You can add bookmarks directly in your Git repository. The extension will pick them up on the next sync and normalize them automatically.

File Format

Create a JSON file with any name in one of the bookmark folders:

{
  "title": "Example Website",
  "url": "https://example.com"
}

Available root folders in the repository:

bookmarks/toolbar/Bookmarks Bar
bookmarks/other/Other Bookmarks
bookmarks/menu/Bookmarks Menu (Firefox)
bookmarks/mobile/Mobile Bookmarks

On the next sync, the extension will rename the file to its canonical format and add it to the folder's ordering.

GitHub Action

A GitHub Action workflow is included to add bookmarks via the GitHub UI or CLI. Copy .github/workflows/add-bookmark.yml to your bookmark repository.

Usage via GitHub CLI

gh workflow run add-bookmark.yml \
  -f url="https://example.com" \
  -f title="Example Website" \
  -f folder="toolbar"

Usage via GitHub Web UI

Go to your repository ‚Üí Actions ‚Üí "Add Bookmark" ‚Üí Run workflow. Fill in the URL, title, and target folder.
