Synchronizacja zakładek przez GitHub. Synergia Linkwarden, Smart Search i aplikacja Companion. Bezpośrednio, bezpiecznie, prywatnie.

### Szczegółowy opis
GitSyncMarks to profesjonalne rozwiązanie do automatycznej i dwukierunkowej synchronizacji zakładek z repozytorium GitHub. Zarządzaj danymi na komputerze przez Chrome i Firefox lub w podróży dzięki aplikacji GitSyncMarks Companion. Bez pośredników, bez serwerów zewnętrznych – pełna kontrola i prywatność.

Highlights

- Synergia Linkwarden: Zapisuj strony lub linki bezpośrednio w swojej instancji Linkwarden. Zawiera automatyczne zrzuty ekranu, synchronizację kolekcji i predefiniowane tagi.
- Smart Search: Dedykowany, błyskawiczny interfejs wyszukiwania zakładek. Obsługuje tryb jasny i ciemny oraz jest w pełni dostępny za pomocą klawiatury.
- Kreator Konfiguracji: Proces wdrażania krok po kroku prowadzi nowych użytkowników przez konfigurację tokena i repozytorium aż do pierwszej udanej synchronizacji.
- Merge Trójdrożny: Niezawodność klasy przemysłowej dzięki zaawansowanemu algorytmowi łączenia, który inteligentnie scala zmiany z wielu urządzeń.

Kluczowe możliwości

- Prywatność z założenia: Bezpośrednia komunikacja z API GitHub. Żadne osoby trzecie nie widzą Twoich danych.
- Przechowywanie w plikach: Każda zakładka jest zapisywana jako czytelny plik JSON – idealny do wersjonowania i ręcznej edycji na GitHubie.
- Wiele profili: Zarządzaj maksymalnie 10 oddzielnymi profilami (np. Praca/Prywatne) z indywidualnymi repozytoriami.
- Automatyzacja: Dodawaj zakładki przez CLI lub GitHub Actions; rozszerzenie zintegruje je automatycznie podczas następnej synchronizacji.
- Generowane pliki: Automatycznie tworzy README.md (przegląd), bookmarks.html (import do przeglądarki) lub kanał RSS w Twoim repozytorium.
- Design i i18n: Profesjonalne motywy jasne i ciemne oraz wsparcie dla 12 języków.

Instalacja i konfiguracja

1. Instalacja: Pobierz plik ZIP z GitHuba (Chrome/Edge) lub zainstaluj z web store.
2. GitHub PAT: Utwórz osobisty token dostępu z uprawnieniami "repo".
3. Kreator konfiguracji: Postępuj zgodnie z instrukcjami w Pomoc -> Pierwsze kroki.
4. Linkwarden (opcjonalnie): Skonfiguruj swoją instancję w dedykowanej zakładce Linkwarden.

Aplikacja Companion
Używaj GitSyncMarks-App (Android, iOS, Desktop), aby zarządzać zakładkami bezpośrednio z repozytorium GitHub na urządzeniach mobilnych.
