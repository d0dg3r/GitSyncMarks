### Name
GitSyncMarks

### Summary (max 250 characters)
Синхронизация закладок через GitHub. Интеграция Linkwarden, умный поиск и мастер настройки. Безопасно, приватно. Полная поддержка Firefox.

### Detailed Description
GitSyncMarks — это профессиональное расширение для Firefox, которое позволяет автоматически синхронизировать закладки с GitHub. Без посредников — ваши данные остаются полностью под вашим контролем.

Основные возможности

- Интеграция с Linkwarden: Сохраняйте страницы напрямую в ваш Linkwarden. Включает автоматическое создание скриншотов, синхронизацию коллекций и тегов.
- Умный поиск (Smart Search): Специализированный, молниеносный интерфейс поиска закладок. Поддерживает темы (светлую/темную) и управление с клавиатуры.
- Мастер настройки: Пошаговое руководство помогает новым пользователям настроить токен и репозиторий для первой успешной синхронизации.
- Трехстороннее слияние (Three-Way Merge): Продвинутый алгоритм предотвращает конфликты версий и интеллектуально объединяет изменения с различных устройств.

Ключевые функции

- Приватность (Private-by-Design): Прямая связь с API GitHub. Третьи лица не имеют доступа к вашим данным.
- Оптимизировано для Firefox: Поддерживает нативную структуру закладок.
- Отдельные файлы: Каждая закладка — это JSON-файл, идеально подходящий для загрузки в GitHub.
- Несколько профилей: Управляйте до 10 профилей (работа, личное) со своими репозиториями.
- Автоматизация: Добавляйте закладки через CLI или GitHub Actions; расширение автоматически интегрирует их.
- Генерация файлов: Автоматически создает README.md, bookmarks.html или RSS-канал в вашем репозитории.

Мобильное приложение
Используйте GitSyncMarks-App (Android, iOS) для управления закладками напрямую с телефона. (Примечание: Firefox на Android не поддерживает прямую синхронизацию закладок расширений — используйте наше приложение).
