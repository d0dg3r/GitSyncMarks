### Название
GitSyncMarks

### Описание (макс. 250 символов)
Синхронизация закладок Firefox через GitHub. Интеграция с Linkwarden, умный поиск и мобильное приложение. Двусторонняя, надежная и приватная защита данных. Полная поддержка Firefox и нативной структуры меню. Без посредников.

### Полное описание
GitSyncMarks — это профессиональное расширение для Firefox, обеспечивающее автоматическую двустороннюю синхронизацию закладок с вашим репозиторием GitHub. Используйте мобильное приложение GitSyncMarks-App для управления данными в пути. Никаких посредников или сторонних серверов — полный контроль и приватность.

Особенности

- Синергия с Linkwarden: сохраняйте страницы и ссылки напрямую в Linkwarden. Автоматические скриншоты, синхронизация коллекций и предустановленные теги.
- Умный поиск: сверхбыстрый интерфейс поиска закладок. Поддержка светлой и темной тем, полное управление с клавиатуры.
- Мастер настройки: пошаговое руководство поможет настроить токен, репозиторий и выполнить первую синхронизацию.
- Трехстороннее слияние (3-Way Merge): продвинутый алгоритм объединяет изменения с разных устройств, обеспечивая промышленный уровень надежности.

Ключевые функции

- Приватность по умолчанию: прямое взаимодействие с GitHub API. Никто, кроме вас, не видит ваши данные.
- Оптимизация для Firefox: поддержка нативной структуры закладок (панель, меню и другие).
- Файловое хранение: каждая закладка сохраняется как читаемый JSON-файл. Идеально для версионного контроля и редактирования.
- Мультипрофильность: поддержка до 10 независимых профилей с разными репозиториями.
- Автоматизация: добавляйте закладки через CLI или GitHub Actions; расширение автоматически подхватит их.
- Генерация файлов: автоматическое создание README.md (обзор), bookmarks.html (для импорта) и RSS-ленты в репозитории.

Установка и настройка

1. Установка: установите GitSyncMarks из магазина Firefox Add-ons (AMO).
2. GitHub PAT: создайте персональный токен доступа с правами «repo».
3. Мастер настройки: следуйте инструкциям в разделе «Помощь -> Быстрый старт».
4. Linkwarden (опционально): настройте параметры экземпляра на вкладке Linkwarden.

Мобильное приложение
Используйте GitSyncMarks-App (Android, iOS, Desktop) для управления закладками в репозитории GitHub прямо с мобильного устройства. (Примечание: Firefox на Android не поддерживает прямую синхронизацию закладок расширениями, используйте наше приложение вместо этого).
