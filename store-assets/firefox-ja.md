# Firefox Add-ons (AMO) — GitSyncMarks (日本語)

### Name
GitSyncMarks

### Summary (max 250 characters)
GitHub経由のブックマーク同期。Linkwarden連携、スマート検索、セットアップウィザード。双方向同期可能でFirefoxメニューにもネイティブ対応。

### Detailed Description
GitSyncMarksは、FirefoxのブックマークをGitHubリポジトリと安全に自動同期するためのプロフェッショナルな拡張機能です。 サードパーティのサーバーなしで、データを完全に制御できます。

ハイライト機能

- Linkwarden連携：ページを直接Linkwardenに保存します。自動スクリーンショット、コレクション同期、タグが含まれます。
- スマート検索：超高速な専用の検索インターフェース。ライト/ダークテーマをサポートし、キーボード操作で完結します。
- セットアップウィザード： 初期設定、トークン、リポジトリの構成を一歩ずつガイドし、最初の同期までをスムーズに導きます。
- 3ウェイマージ: 複数のデバイスからの変更をインテリジェントに組み合わせる高度なマージアルゴリズム。

主な特徴

- プライベート・バイ・デザイン： GitHub APIとの直接通信のみ。第三者がデータを見ることはありません。
- Firefox最適化： 標準のツールバーなどの内部構造をサポートします。
- 単一ファイル保存： 各ブックマークは読みやすいJSONファイルとして保存されます。
- 複数プロファイル： 仕事とプライベートなどで最大10個のリポジトリセットを分離・管理できます。
- 自動化： CLI等からブックマークをリモート追加しても、次の拡張機能の同期時に自動的に統合・取り込みされます。
- 生成ファイル： README.md、インポート用HTML、またはRSSフィードを生成できます。

コンパニオンアプリ
モバイル上ではGitSyncMarks-Appを活用してください。（注: Android版FirefoxはAPI制約のため拡張機能での同期がサポートされていません。）
