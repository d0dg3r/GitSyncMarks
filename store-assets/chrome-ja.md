### Summary (max 132 characters)
GitHub経由のブックマーク同期。Linkwarden連携、スマート検索、コンパニオンアプリ付。直接、安全、プライベート。

### Detailed Description
GitSyncMarksは、ブックマークをGitHubリポジトリと自動的かつ双方向に同期するプロフェッショナルな拡張機能です。仲介サーバーを使わず、完全な制御とプライバシーを提供します。

ハイライト機能

- 同期履歴と復元: 過去のコミットを閲覧し、差分プレビューで変更を確認し、ワンクリックで以前のブックマーク状態を復元できます。
- 重複修正: 同名フォルダが同期のたびに増殖しなくなりました — 重複は自動的に統合されます。
- Linkwarden連携：ページを直接Linkwardenに保存します。自動スクリーンショット、コレクション同期、タグが含まれます。
- スマート検索：超高速な専用の検索インターフェース。ライト/ダークテーマをサポートし、キーボード操作で完結します。
- セットアップウィザード： 初期設定、トークン、リポジトリの構成を一歩ずつガイドし、最初の同期までをスムーズに導きます。

主な特徴

- プライベート・バイ・デザイン： GitHub APIとの直接通信のみ。第三者がデータを見ることはありません。
- 単一ファイル保存： 各ブックマークは人間が読みやすいJSONファイルとして保存され、バージョン管理に最適です。
- 複数プロファイル： 仕事とプライベートなどで最大10個のリポジトリセットを分離・管理できます。
- 自動化： CLI等からブックマークをリモート追加しても、次の拡張機能の同期時に自動的に統合・取り込みされます。
- などの生成ファイル： README.md、インポート用HTML、またはRSSフィードを生成させることができます。
- Design & i18n: ライト・ダーク・自動テーマ、UI密度調整（コンパクト / ミディアム / ラージ）を設定・ポップアップ・検索・Linkwarden保存で対応、12言語。
