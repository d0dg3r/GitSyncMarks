### Summary (max 132 characters)
Синхр. закладок в ваш Git-репозиторий. Linkwarden, Smart Search, Companion App. Напрямую, безопасно, приватно.

### Detailed Description
GitSyncMarks двусторонне синхронизирует закладки с вашим Git-репозиторием — крупные хостинг-платформы и self-hosted Git-серверы. Управляйте данными на десктопе в вашем браузере или в пути с GitSyncMarks Companion App. Без посредников, без сторонних серверов – полный контроль и конфиденциальность.

Основное

- Git-синхронизация с несколькими провайдерами: Подключите свой Git-хост или self-hosted сервер — каждый профиль использует свой провайдер и URL. Полный список провайдеров: https://github.com/d0dg3r/GitSyncMarks/blob/main/docs/PROVIDERS.md
- Перенос профилей и push-зеркала: Копирование закладок между профилями (замена или слияние); опциональные push-only резервные remotes после каждой синхронизации.
- Прогресс синхронизации в реальном времени: Текст шага при push, pull и смене профиля (напр. `3 / 12 файлов` или `1 из 3` шагов).
- Резервное копирование, совместимое с Bitwarden, в Git: Хранение экспортов хранилища с паролем в репозитории, опциональное доп. шифрование; список, загрузка или удаление удалённых резервных копий.
- UI nested-card: Более понятные группированные секции в настройках, мастере, popup и поиске.
- История синхронизации и восстановление: Просмотр прошлых коммитов, предпросмотр изменений через diff и восстановление любого предыдущего состояния одним кликом.
- Очистка удалённых сирот: Предпросмотр и удаление удалённых файлов закладок, которых больше нет локально.
- Синергия Linkwarden: Сохранение страниц или ссылок в экземпляр Linkwarden — скриншоты viewport, синхронизация коллекций и предопределённые теги.
- Smart Search: Выделенный молниеносный поиск закладок со светлой/тёмной темой и полной клавиатурной навигацией.
- Мастер настройки: Проверка подключения только валидирует доступ; вы выбираете pull, merge/sync, push, настройку папок или пропуск — с подтверждением перед записью в репозиторий.
- Производительность self-hosted Git: Быстрое чтение git tree + blob и single-commit push на совместимых хостах (fallback Contents API при необходимости).
- Контекстное меню: Быстрые папки, popup поиска закладок, Открыть всё из папки, копирование/загрузка favicon и действия профиля по правому клику.
- Синхронизация настроек в Git: Зашифрованная резервная копия настроек (`settings.enc`) в репозитории — общая конфигурация между устройствами.

Ключевые возможности

- Конфиденциальность by design: Прямое общение с API Git-провайдера. Никакие третьи лица не видят ваши данные.
- Трёхстороннее слияние: Промышленная синхронизация автоматически обрабатывает одновременные изменения на нескольких устройствах.
- Хранение по файлам: Каждая закладка — читаемый JSON-файл – идеально для версионирования и ручного редактирования.
- Несколько профилей: До 10 отдельных профилей для работы, личного использования или проектов, каждый со своим репозиторием.
- Автоматизация: Добавляйте JSON закладок в репозиторий через git или с помощью входящего в комплект шаблона GitHub Action; расширение импортирует их при следующей синхронизации.
- Генерируемые файлы: README.md (обзор), bookmarks.html (импорт браузера), RSS-лента и dashy-conf.yml — опционально для каждого файла.
- Дизайн и i18n: Светлая, тёмная и автоматическая темы; настраиваемая плотность UI (компактный / средний / большой); 12 языков.
