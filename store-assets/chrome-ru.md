### Название
GitSyncMarks

### Описание (макс. 132 символа)
Синхронизация закладок через GitHub. Интеграция с Linkwarden, умный поиск и мобильное приложение. Прямо, надежно, приватно.

### Полное описание
GitSyncMarks — это профессиональное решение для автоматической двусторонней синхронизации закладок с вашим репозиторием GitHub. Управляйте данными на десктопе (Chrome, Firefox) или используйте мобильное приложение GitSyncMarks-App для быстрого доступа в пути. Никаких посредников или сторонних серверов — полный контроль и приватность.

Особенности

- Синергия с Linkwarden: сохраняйте страницы и ссылки напрямую в Linkwarden. Автоматические скриншоты, синхронизация коллекций и предустановленные теги.
- Умный поиск: сверхбыстрый интерфейс поиска закладок. Поддержка светлой и темной тем, полное управление с клавиатуры.
- Мастер настройки: пошаговое руководство поможет новым пользователям настроить токен, репозиторий и выполнить первую синхронизацию.
- Трехстороннее слияние (3-Way Merge): продвинутый алгоритм объединяет изменения с разных устройств, обеспечивая промышленный уровень надежности.

Ключевые функции

- Приватность по умолчанию: прямое взаимодействие с GitHub API. Никто, кроме вас, не видит ваши данные.
- Файловое хранение: каждая закладка сохраняется как читаемый JSON-файл. Идеально для версионного контроля и ручного редактирования на GitHub.
- Мультипрофильность: поддержка до 10 независимых профилей (например, «Работа» и «Личное») с разными репозиториями.
- Автоматизация: добавляйте закладки через CLI или GitHub Actions; расширение автоматически подхватит их при следующей синхронизации.
- Генерация файлов: автоматическое создание README.md (обзор), bookmarks.html (для импорта) и RSS-ленты в вашем репозитории.
- Профессиональный дизайн: 12 поддерживаемых языков и элегантный интерфейс.

Установка и настройка

1. Установка: скачайте ZIP с GitHub (для Chrome/Edge) или установите из магазина приложений.
2. GitHub PAT: создайте персональный токен доступа с правами classic «repo» или fine-grained «Contents: Read/Write».
3. Мастер настройки: следуйте инструкциям в разделе «Помощь -> Быстрый старт».
4. Linkwarden (опционально): настройте параметры экземпляра на вкладке Linkwarden.

Мобильное приложение
Используйте GitSyncMarks-App (Android, iOS, Desktop) для управления закладками в репозитории GitHub прямо с мобильного устройства.
