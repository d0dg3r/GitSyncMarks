### Name

### Summary (max 132 characters)
Синхронизация закладок через GitHub. Интеграция с Linkwarden, Smart Search и приложение-компаньон. Напрямую, безопасно, приватно.

### Detailed Description
GitSyncMarks — это профессиональное решение для двунаправленной и автоматической синхронизации закладок с репозиторием GitHub. Без посредников, без сторонних серверов — полный контроль и конфиденциальность.

Основные возможности

- История синхронизации и восстановление: Просматривайте прошлые коммиты, предварительно оценивайте изменения через diff и восстанавливайте любое предыдущее состояние закладок в один клик.
- Исправление дубликатов: Папки с одинаковыми именами больше не размножаются при синхронизации — дубликаты объединяются автоматически.
- Интеграция с Linkwarden: Сохраняйте страницы напрямую в ваш Linkwarden. Включает автоматическое создание скриншотов, синхронизацию коллекций и тегов.
- Умный поиск (Smart Search): Специализированный, молниеносный интерфейс поиска закладок. Поддерживает темы (светлую/темную) и управление с клавиатуры.
- Мастер настройки: Пошаговое руководство помогает новым пользователям настроить токен и репозиторий для первой успешной синхронизации.

Ключевые функции

- Приватность (Private-by-Design): Прямая связь с API GitHub. Третьи лица не имеют доступа к вашим данным.
- Отдельные файлы: Каждая закладка сохраняется как понятный JSON-файл — идеально для контроля версий в GitHub.
- Несколько профилей: Управляйте до 10 профилей (работа, личное) со своими репозиториями.
- Автоматизация: Добавляйте закладки через CLI или GitHub Actions; расширение автоматически интегрирует их.
- Генерация файлов: Автоматически создает README.md, bookmarks.html или RSS-канал в вашем репозитории.
- Design & i18n: Светлая, тёмная и системная темы; настраиваемая плотность интерфейса (компактная / средняя / крупная) в настройках, popup, поиске и сохранении Linkwarden; 12 языков.

Мобильное приложение
Используйте GitSyncMarks-App (Android, iOS) для управления закладками прямо из репозитория GitHub на мобильных устройствах.
